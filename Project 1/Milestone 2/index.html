Experience


I Need Space
I Need Space is a singleplayer game developed in the Unity game engine for the LA Game Jam(2022). I worked on the game as a programmer and implemented the user interface alongside some of the character animations. You play as two characters at the same time, but the catch is that they have inverted movement mechanics.

Vehicular Cloud Real Time System (VCRTS)
VCRTS is a system simulation that manages computational resources and jobs into a database. I helped to program the user interface of the system by using JSwing and Java. Furthermore, I also aided in implementing the database collection using MySQL.


Synford Text-Based RPG
Synford is a singleplayer game programmed in python and is played through the terminal. This was a solo project I had started in 2022 in order to learn more about Python. It includes a working inventory system as well as turn-based combat and randomized drops.
